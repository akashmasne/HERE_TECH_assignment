# HERE_TECH_assignment
HERE_TECH_assignment

Process development challenge
1. Subscribe for a freemium developer account (https://developer.here.com/sign-up?create=Freemium-Basic&keepState=true&step=account)
2. Collect an address sample set of at least 100 addresses as 10 from 10 different cities
3. write a Python application that 
  Reads the above addresses from a text file
  Query the Geocoding & Search API by the geocoding endpoint to retrieve the results
  Write the results into a csv file with reference to the queried address
  Measure quality (as defined below) per total, city
4. Quality assessment challenge:
  Define a quality metric
  Design a process for measurement
  Design a reporting dashboard
5. Present this (powerpoint or alike)
